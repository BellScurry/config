My Config
==============================================
##How to install?

Clone this repository into local.
```
$ git clone https://github.com/BellScurry/config
```

Installation
```
$ ~/config/install.sh
```

##This repository consists of

* Shell Settings
  * .profile
  * .bashrc
* Vim Settings
  * .vimrc
  * .vim
* Git Settings
  * .gitconfig

###Vim Plugins

* Vundle
* NerdTree
* Several Color Schemes
